Basic Fitness Plan — Phone App / PWA

This is a simple offline-capable web app for the Tier 1 six-week workout routine.

Files:
- index.html: the app
- manifest.webmanifest: install metadata
- service-worker.js: offline cache
- icons/: app icons

Use options:
1. Open index.html directly in a browser for basic use.
2. For true phone installation, upload this folder to GitHub Pages or any HTTPS host, open the URL on your phone, then choose Add to Home screen / Install app.

Data:
- Logs are stored locally in the browser with localStorage.
- Nothing leaves the device.
- Clearing browser/site data will clear the log.
